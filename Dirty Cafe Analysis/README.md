# Dirty Cafe Data Analysis

## Overview
This project focuses on analyzing operational and sales data from Dirty Cafe to uncover patterns, customer behavior, and performance insights.

The goal is to transform raw data into meaningful information that can support better business decisions.

## Dataset
The dataset includes transactional and operational cafe data such as Item, Quantity, Price Per Unit, Total Spent, Payment Method, Location,	Transaction Date.

Data cleaning was performed to handle missing values, formatting issues, and inconsistencies before analysis.

## Analysis Workflow

1. Data cleaning and preprocessing
2. Exploratory data analysis (EDA)
3. Trend and behavior analysis
4. Visualization of key insights
5. Interpretation of results

## Key Insights

- Identification of peak business hours
- Popular menu items and demand patterns
- Revenue trends
- Customer ordering behavior

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Conclusion

The analysis highlights actionable insights that can help improve operational efficiency, menu strategy, and customer experience.

This project demonstrates practical data analysis skills including cleaning, exploration, visualization, and insight extraction.
