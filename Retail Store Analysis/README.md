# Retail Store Data Analysis

## Overview
This project analyzes retail store transactional data to uncover purchasing patterns, customer behavior, and sales performance trends.

The objective is to transform raw retail data into actionable insights that support smarter business decisions and revenue optimization.

## Dataset
The dataset includes retail transactions with the following attributes:
Transaction ID, Customer ID, Category, Item, Price Per Unit, Quantity,
Total Spent, Payment Method, Location, Transaction Date, Discount Applied.

Data cleaning was performed to handle missing values, formatting issues,
and inconsistencies before analysis.

## Analysis Workflow

1. Data cleaning and preprocessing
2. Exploratory data analysis (EDA)
3. Sales and category performance analysis
4. Customer purchasing behavior analysis
5. Visualization of key trends
6. Insight interpretation

## Key Insights

- Top performing product categories
- Revenue contribution by location
- Customer purchasing patterns
- Payment method preferences
- Seasonal and time-based sales trends

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Conclusion

The analysis reveals patterns that can improve sales strategy, inventory planning,
and customer targeting.

This project demonstrates practical data analysis skills including cleaning,
exploration, visualization, and extracting meaningful business insights.
